# Dynamics and Static Structure of Microswimmers under a Constant External Field

## Abstract

In this thesis, a detailed computational study was conducted on the dynamics and structure of a microscopic system of a finite number of unconfined dipolar microswimmers modeled through Brownian dynamics equations under the influence of a constant external magnetic field. The equations of motion were solved using the Euler-Maruyama method. Five structural order parameters were calculated, and seven possible final states of the system were identified, summarized in a state diagram in terms of the initial dipolar interaction strengths and the coupling with the external field. The analysis of the structure factor reveals states for which structural relaxation does not occur, as well as its relationship with the intensity of interactions between microswimmers. On the other hand, the analysis of dynamic properties such as the mean squared displacement shows that translational motion remains in the superdiffusive regime, transitioning towards ballistic motion in the presence of an external field. Furthermore, rotational dynamics remains in a subdiffusive regime characteristic of the suppression of rotational diffusion mediated by the external field.

*Keywords*: _microswimmer, Brownian dynamics, magnetic dipole, external field, active matter._

## Repository Contents

* `analysis.ipynb`: Jupyter Notebook containing code cells that invoke the main functions to generate, analyze, and visualize simulation data.
* `final_functions.py`: Python module containing the complete set of functions used for data generation, processing, analysis, and visualization related to the Brownian Dynamics simulations.
